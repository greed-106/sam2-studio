# 视频掩码生成教程

本文演示如何打开一个视频，对其中一个物体生成 mask，传播到所有帧，并导出二值 PNG mask。图片序列目录也可以按同样方式处理：点击 `打开图片目录`，目录内图片会按自然排序作为帧序列。

## 1. 启动软件

```bash
uv run sam2-studio
```

## 2. 选择模型

点击 `选择模型`，选择本地 SAM2/SAM2.1 权重，例如：

```text
sam2.1_l.pt
```

如果你还没有手动点击 `加载模型`，第一次添加点/框 prompt 时软件也会自动排队加载模型。

## 3. 打开视频

点击 `打开媒体`，选择 `.mp4`、`.avi`、`.mkv`、`.mov` 或 `.webm` 视频。

打开后可以使用底部播放条：

- `播放/暂停`：播放或暂停。
- `上一帧` / `下一帧`：逐帧切换，点击后会暂停当前播放。
- slider：跳转到指定帧。
- `P:x M:y`：当前项目中有 prompt 的帧数和有 mask 的帧数。

## 4. 选择对象

默认已有 `object 0`。如果要处理多个物体，点击 `新增对象` 创建新对象，并在对象下拉框中切换。

## 5. 在关键帧添加 prompt

找到物体清晰的一帧，暂停播放。

推荐流程：

1. 点击 `框选`，拖一个框覆盖目标物体。
2. 如果 mask 不够准，点击 `正点` 在目标内部点一下。
3. 如果误分割了背景，点击 `负点` 在错误区域点一下。

点/框 prompt 会触发当前帧 SAM2 分割。终端第一次可能打印 Ultralytics 设备信息，这是模型初始化日志。

## 6. 传播到所有帧

如果当前帧位于视频中间，点击 `先正后反传播`。

含义：

- 先从当前帧传播到视频末尾。
- 完成后再用同一份 prompt/object/frame seed 传播回第 0 帧。
- 传播结果会写入每一帧对应 object 的 mask。

如果当前帧已经是第 0 帧，可以只点击 `正向传播▶`。如果只想补当前帧之前的帧，点击 `反向传播◀`。

## 7. 检查和修正

传播结束后拖动 slider 检查不同帧。

如果某一帧结果不理想：

- 可以在该帧添加新的点/框 prompt，重新分割当前帧。
- 也可以用 `多边形` 直接画一个 mask。
- 也可以用 `笔刷+` / `笔刷-` 手动修补。

注意：多边形和笔刷是手工 mask 编辑，不会作为视频传播 seed。如果要重新传播，请使用点或框 prompt。

## 8. 保存项目

点击 `保存项目`，保存为 `.sam2studio/` 项目目录。

项目保存后可以恢复媒体引用、对象列表、点/框 prompt、已传播或手工编辑的 mask，以及模型路径和参数。

## 9. 导出 mask

根据用途选择导出方式：

- 只导出当前帧：`导出当前二值掩码`。
- 导出传播后的所有已有帧：`导出全部二值PNG`。
- 只想给人看效果图：`导出当前叠加图`。

传播完成后，如果你想要整段视频的 mask，不要使用 `导出当前二值掩码`，它只导出当前帧。应使用 `导出全部二值PNG`。

## 10. 常见问题

### 为什么终端打印两次 Ultralytics banner？

当前帧分割和视频传播使用不同 predictor。框选后当前帧分割会初始化 `SAM2Predictor`，传播会初始化 `SAM2VideoPredictor`，所以可能看到两次设备信息。这是预期行为。

### 传播后显存为什么没有完全释放？

视频传播 predictor 会在任务结束后释放并清理 CUDA cache。但当前帧分割模型会保留，以便后续点击更快。PyTorch 也可能保留一部分 CUDA cache，所以系统工具看到的显存不一定马上归零。
