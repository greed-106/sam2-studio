# 功能说明

## 定位

SAM2 Studio 是掩码生成器。它帮助用户从单张图片、视频或图片序列目录中生成 mask，并导出二值 PNG、NPZ、safetensors 或叠加预览图。它暂时不提供数据集标注格式导出。

## 主界面区域

- 顶部工具栏：打开媒体、选择/加载模型、添加 prompt、传播、导出和项目保存。
- 模型路径栏：显示当前选择的 SAM2/SAM2.1 `.pt` 权重路径。
- 中央画布：显示当前图片、视频帧或图片序列帧、prompt 和 mask overlay。
- 播放条：视频和图片序列播放、暂停、上一帧、下一帧、seek，并显示提示帧和 mask 帧统计。
- 状态栏：显示加载、分割、传播、导出等状态。

## 按钮功能

- `打开媒体`：打开单张图片或视频文件。
- `打开图片目录`：选择一个目录，按自然排序读取其中所有图片，并把它们当作视频抽帧序列处理。
- `加载模型`：加载模型路径栏中的 SAM2/SAM2.1 权重，用于当前帧分割。
- `选择模型`：选择本地 `.pt` 或 `.pth` 模型文件。
- `正点`：给当前对象添加前景点。
- `负点`：给当前对象添加背景点。
- `框选`：拖拽矩形框作为 SAM2 box prompt。
- `多边形`：左键添加顶点，右键或双击闭合；闭合后直接生成当前对象 mask，不作为传播 seed。
- `笔刷+`：在当前对象 mask 上增加区域。
- `笔刷-`：从当前对象 mask 中擦除区域。
- `撤销`：优先撤销最近一次 mask 编辑；如果没有 mask 编辑，则撤销当前帧当前对象的最后一个 prompt。
- `新增对象`：创建新的对象 id 和颜色，并切换为当前对象。
- `正向传播▶`：从当前帧传播到视频或图片序列最后一帧。
- `反向传播◀`：从当前帧倒序传播到视频或图片序列第 0 帧。
- `先正后反传播`：先执行正向传播，再使用同一份 prompt/object/frame seed 执行反向传播。
- `取消传播`：请求取消正在运行的传播。取消是协作式的，可能要等当前 GPU 调用返回后生效。
- `导出当前二值掩码`：导出当前帧可见对象的 mask。PNG 为黑白二值图，背景 `0`，前景 `255`。
- `导出全部二值PNG`：导出所有已有 mask 的帧，每帧一个黑白二值 PNG，并写 `metadata.json`。
- `导出当前叠加图`：导出当前帧图像和可见 mask 的可视化叠加 PNG，不适合作为训练 mask。
- `保存项目`：保存 `.sam2studio/` 项目目录。
- `加载项目`：恢复 `.sam2studio/` 项目。

## 分割和传播

- 当前帧分割使用 `SAM2Predictor`，用于点/框 prompt 后生成当前帧 mask。
- 视频和图片序列传播使用 `SAM2VideoPredictor`，会沿帧序列生成 mask。
- 这两个 predictor 不同，因此终端可能分别打印一次 Ultralytics 设备信息。
- 传播 seed 只能来自当前对象当前帧的点/框 prompt。多边形和笔刷是手动 mask 编辑，不能作为 SAM2VideoPredictor 的 seed。

## 项目保存

`.sam2studio/` 项目目录会保存媒体引用、图片序列帧文件名、模型设置、对象、点/框 prompt 和已有 mask。项目不会保存视频帧像素、图片序列帧像素或模型 embedding，重新打开项目时会从原始媒体重新读取帧。
